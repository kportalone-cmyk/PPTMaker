핵심 기능

기업 환경에 최적화된 AI 문서 생성 솔루션

AI 콘텐츠 생성

LLM 기반 AI가 리소스를 분석하여 문서 콘텐츠를 자동 생성. 스트리밍으로 실시간 확인 가능.

커스텀 템플릿

기업 CI/BI 가이드에 맞는 템플릿을 관리자가 직접 구축. 배경, 폰트, 레이아웃을 자유롭게 설계.

웹 검색 통합

Perplexity API 연동으로 실시간 웹 검색 결과를 리소스로 활용. 최신 정보 기반 문서 생성.

실시간 협업

SSE 기반 실시간 동기화. 슬라이드 잠금, 온라인 상태 표시, 변경 이력 관리 지원.

기업 보안

JWT 기반 인증, 조직도 SSO 연동, 암호화 처리. 다중 서버 L4 이중화 환경 지원.

다국어 번역

한국어, 영어, 일본어, 중국어 4개 언어 지원. 생성된 문서를 클릭 한 번으로 번역.

관리자 기능

템플릿부터 프롬프트까지, 관리자가 완전히 통제

템플릿 빌더

  • 캔버스 기반 드래그&드롭 슬라이드 편집기
  • 배경 이미지, 텍스트/이미지 오브젝트 배치
  • 폰트, 색상, 크기 세부 설정 (서버 등록 폰트)
  • 슬라이드 메타정보 (제목/거버넌스/서브텍스트) 관리

AI 프롬프트 관리

  • 문서 타입별 시스템/사용자 프롬프트 커스터마이징
  • 생성 품질 및 스타일 제어 (톤, 문체, 구조)
  • 아웃라인 모델 및 콘텐츠 모델 분리 설정
