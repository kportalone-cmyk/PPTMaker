# OfficeMaker - AI 기반 기업용 문서 자동 생성 솔루션

---

## 1. 제품 개요

**OfficeMaker**는 기업 환경에 최적화된 AI 기반 문서 자동 생성 플랫폼입니다.

기업의 브랜드 템플릿과 내부 리소스를 활용하여, AI가 프레젠테이션(PPTX), 스프레드시트(XLSX), 워드 문서(DOCX)를 자동으로 생성합니다. 관리자가 구축한 기업 전용 템플릿 위에 사용자가 등록한 자료를 바탕으로 전문적인 문서를 수 분 안에 완성할 수 있습니다.

> **"자료만 넣으면, AI가 기업 브랜드에 맞는 문서를 만들어 드립니다."**

---

## 2. 핵심 컨셉

### 2.1 기업 맞춤형 AI 문서 생성

OfficeMaker는 범용 AI 도구와 달리, **기업 고유의 템플릿과 디자인 가이드라인**을 기반으로 문서를 생성합니다.

- 관리자가 기업 CI/BI에 맞는 슬라이드 템플릿을 직접 구축
- AI가 템플릿의 레이아웃, 텍스트 영역, 이미지 배치를 인식하여 콘텐츠를 자동 배치
- 생성된 문서는 별도 디자인 작업 없이 바로 활용 가능

### 2.2 리소스 기반 지능형 콘텐츠 생성

단순한 키워드 입력이 아닌, **실제 업무 자료를 기반으로 문서를 생성**합니다.

- 기존 보고서, 데이터, 회의록 등을 리소스로 등록
- AI가 리소스를 분석하여 핵심 내용을 추출하고 구조화
- 웹 검색을 통한 최신 정보 자동 보강

### 2.3 기업 인프라 통합

기존 기업 시스템과 자연스럽게 연동됩니다.

- 조직도/인사 시스템 연동 (LDAP/AD)
- SSO(Single Sign-On) 지원
- 다중 서버(L4) 이중화 환경 완벽 대응

## 3. 주요 기능

### 3.1 AI 문서 자동 생성

| 기능 | 설명 |
|------|------|
| **프레젠테이션 생성** | 기업 템플릿 기반 PPTX 자동 생성. 표지 → 목차 → 섹션 → 본문 → 마무리 슬라이드 구조 자동 설계 |
| **스프레드시트 생성** | 데이터 분석 결과를 표, 차트(Bar/Line/Pie) 포함 XLSX로 자동 생성 |
| **워드 문서 생성** | 보고서, 기획서 등 서식 있는 DOCX 문서 자동 생성 (제목/본문/표/차트 포함) |
| **실시간 스트리밍** | AI가 콘텐츠를 생성하는 과정을 실시간 타이핑 효과로 확인 가능 |
| **스마트 레이아웃** | AI가 템플릿 제약조건(텍스트 영역 크기, 이미지 위치 등)을 인식하여 최적의 콘텐츠 배치 |

### 3.2 관리자 템플릿 관리

관리자가 기업 전용 슬라이드 템플릿을 직접 구축하고 관리합니다.

- **비주얼 캔버스 편집기**: 드래그 앤 드롭 방식의 직관적 슬라이드 편집
- **오브젝트 관리**: 텍스트/이미지/도형 오브젝트의 위치, 크기, 스타일 자유 설정
- **텍스트 스타일링**: 폰트, 색상, 크기, 정렬 등 세밀한 텍스트 속성 제어
- **배경 이미지 관리**: 공통 배경 및 슬라이드별 개별 배경 설정
- **슬라이드 유형 분류**: 표지, 목차, 섹션 구분, 본문, 마무리 등 슬라이드 역할 지정
- **메타정보 설정**: 소제목 수, 설명 수 등 AI 생성 시 참고할 메타데이터 구성
- **폰트 등록**: 기업 전용 폰트 서버 등록 및 관리
- **템플릿 발행 관리**: 제작 완료된 템플릿만 사용자에게 공개

### 3.3 리소스 관리

다양한 형태의 자료를 리소스로 등록하여 AI 문서 생성의 기반 데이터로 활용합니다.

| 리소스 유형 | 지원 형식 | 설명 |
|------------|----------|------|
| **파일 업로드** | Word, Excel, PowerPoint, PDF, Text, CSV | 기존 문서에서 텍스트를 자동 추출하여 활용 |
| **텍스트 입력** | 자유 텍스트 | 직접 내용을 입력하여 리소스로 등록 |
| **웹 검색** | Perplexity AI | 키워드 기반 최신 웹 정보 자동 수집 (최대 10건) |
| **URL 크롤링** | 웹페이지, YouTube | URL 입력 시 본문 자동 추출, YouTube 자막 자동 수집 |

### 3.4 온라인 문서 편집 (OnlyOffice 연동)

AI가 생성한 문서를 브라우저에서 직접 편집할 수 있습니다.

- **실시간 협업 편집**: 여러 사용자가 동시에 같은 문서를 편집
- **PPTX/XLSX/DOCX 지원**: 세 가지 문서 형식 모두 온라인 편집 가능
- **자동 저장**: 편집 내용 자동 저장 및 버전 관리
- **익숙한 UI**: MS Office와 유사한 인터페이스로 학습 비용 최소화

### 3.5 협업 및 공유

| 기능 | 설명 |
|------|------|
| **프로젝트 공유** | Owner / Editor / Viewer 역할 기반 접근 제어 |
| **슬라이드 잠금** | 동시 편집 충돌 방지를 위한 자동 잠금 메커니즘 |
| **공유 링크** | 로그인 없이 접근 가능한 미리보기/다운로드 링크 생성 |
| **접속 현황** | 실시간 온라인 사용자 확인 |

### 3.6 다운로드 및 내보내기

생성된 문서를 다양한 형식으로 다운로드할 수 있습니다.

- **PPTX**: 파워포인트 파일 다운로드 (배경 이미지, 텍스트 스타일링 적용)
- **XLSX**: 엑셀 파일 다운로드 (차트, 스타일링, 자동 열 너비 조정 포함)
- **DOCX**: 워드 문서 다운로드 (서식, 표, 이미지, 차트 포함)
- **PDF**: 프레젠테이션 PDF 변환 다운로드

### 3.7 다국어 지원

글로벌 기업 환경을 위한 다국어 인터페이스를 제공합니다.

- 한국어 (기본)
- 영어
- 일본어
- 중국어
- 관리자 설정으로 지원 언어 구성 가능

## 4. 활용 시나리오

### 4.1 경영 보고서 자동 생성

> **상황**: 매월 경영실적 보고 PPT를 작성해야 하는 경영기획팀

1. 실적 데이터(Excel)를 리소스로 업로드
2. 기업 표준 보고서 템플릿 선택
3. "월간 경영실적 보고" 지침 입력
4. AI가 데이터를 분석하여 차트와 핵심 인사이트가 포함된 보고서 자동 생성

**효과**: 기존 2~3시간 소요 → 10분 내 초안 완성

### 4.2 제안서/기획서 작성

> **상황**: 신규 프로젝트 제안서를 급하게 준비해야 하는 영업팀

1. 관련 자료(기존 제안서, 고객 요구사항 문서)를 리소스로 등록
2. 웹 검색으로 시장 동향, 경쟁사 정보 자동 수집
3. 기업 제안서 템플릿 선택 후 AI 생성
4. OnlyOffice에서 세부 내용 편집 후 바로 제출

**효과**: 제안서 초안 작성 시간 80% 절감

### 4.3 교육/세미나 자료 제작

> **상황**: 사내 교육 자료를 정기적으로 제작하는 인사/교육팀

1. 교육 커리큘럼 텍스트 입력
2. 참고 자료(PDF, 웹 링크) 등록
3. 교육용 템플릿 선택
4. AI가 교육 흐름에 맞는 슬라이드 자동 구성

**효과**: 일관된 품질의 교육 자료를 빠르게 생산

### 4.4 데이터 분석 보고서

> **상황**: 대량 데이터를 분석하여 엑셀 보고서를 만들어야 하는 데이터 분석팀

1. 원본 데이터(CSV, Excel) 업로드
2. "매출 트렌드 분석 및 예측" 지침 입력
3. AI가 데이터를 구조화하고 차트(Bar, Line, Pie)가 포함된 엑셀 보고서 생성
4. 온라인 편집기에서 추가 분석 작업

**효과**: 데이터 정리 및 시각화 작업 자동화

### 4.5 글로벌 문서 제작

> **상황**: 해외 법인에 보낼 보고서를 다국어로 준비해야 하는 글로벌 사업부

1. 한국어 원본 자료 업로드
2. 영어/일본어/중국어 언어 설정으로 전환
3. AI가 해당 언어로 문서 자동 생성
4. 공유 링크로 해외 담당자에게 즉시 전달

**효과**: 번역 + 문서 작성을 한 번에 해결

## 5. 제품 특징 및 차별점

### 5.1 기업 전용 설계

| 항목 | OfficeMaker | 일반 AI 문서 도구 |
|------|----------|------------------|
| 기업 템플릿 | 관리자가 직접 구축한 기업 CI/BI 템플릿 | 범용 템플릿만 제공 |
| 조직도 연동 | LDAP/AD 조직도 직접 연동 | 별도 사용자 관리 필요 |
| SSO 지원 | 기업 SSO(JWT) 자연스러운 통합 | 별도 계정 생성 필요 |
| 보안 | 온프레미스 설치, 데이터 외부 유출 없음 | 클라우드 의존, 데이터 외부 저장 |
| 이중화 | L4 다중 서버 환경 지원 | 단일 서버 구조 |

### 5.2 최신 AI 엔진

- **Anthropic Claude** 최신 모델 탑재 (Opus 4.6 / Sonnet 4.6)
- 용도별 모델 자동 선택 (고품질 생성 vs 빠른 구조화)
- 관리자가 프롬프트와 모델을 직접 커스터마이징 가능
- AI 장애 시 자동 폴백 처리로 서비스 연속성 보장

### 5.3 올인원 문서 플랫폼

하나의 플랫폼에서 세 가지 문서 형식을 모두 지원합니다.

```
┌─────────────────────────────────────────────────────┐
│                    OfficeMaker                          │
│                                                     │
│   📊 프레젠테이션     📈 스프레드시트     📝 워드 문서   │
│   (PPTX)             (XLSX)             (DOCX)      │
│                                                     │
│   ┌───────────────────────────────────────────┐     │
│   │          AI 자동 생성 엔진                  │     │
│   │   리소스 분석 → 구조 설계 → 콘텐츠 생성     │     │
│   └───────────────────────────────────────────┘     │
│                                                     │
│   ┌───────────────────────────────────────────┐     │
│   │          온라인 협업 편집기                  │     │
│   │   실시간 공동 편집 · 자동 저장 · 버전 관리   │     │
│   └───────────────────────────────────────────┘     │
│                                                     │
│   ┌───────────────────────────────────────────┐     │
│   │          다운로드 & 공유                    │     │
│   │   PPTX · XLSX · DOCX · PDF · 공유 링크     │     │
│   └───────────────────────────────────────────┘     │
└─────────────────────────────────────────────────────┘
```

### 5.4 유연한 리소스 활용

- **7종 파일 형식** 자동 텍스트 추출 (Word, Excel, PPT, PDF, Text, CSV)
- **Perplexity AI** 연동 웹 검색으로 최신 정보 자동 수집
- **YouTube 자막** 자동 추출로 영상 콘텐츠도 리소스로 활용
- **URL 크롤링**으로 웹페이지 본문 자동 수집

### 5.5 엔터프라이즈급 안정성

- **비동기 아키텍처**: FastAPI + Motor(비동기 MongoDB 드라이버)로 높은 동시 처리 성능
- **L4 이중화 지원**: 서버 메모리 무상태(Stateless) 설계, 모든 상태는 DB에 저장
- **자동 인덱싱**: MongoDB 성능 최적화를 위한 자동 인덱스 생성
- **커넥션 풀링**: MongoDB/Redis 커넥션 풀 관리로 자원 효율화
- **캐싱**: Redis 기반 사용자 데이터 캐싱 (15분 TTL)

### 5.6 관리자 통제력

관리자가 AI의 동작을 세밀하게 제어할 수 있습니다.

- **프롬프트 편집**: AI에게 전달되는 지시사항을 관리자가 직접 수정
- **모델 선택**: 프롬프트별로 사용할 AI 모델 지정 가능
- **템플릿 발행 관리**: 완성된 템플릿만 사용자에게 공개
- **폰트/배경 관리**: 기업 표준 디자인 요소 중앙 관리

### 6.2 기술 스택

| 영역 | 기술 |
|------|------|
| **백엔드** | Python FastAPI (비동기), Uvicorn |
| **데이터베이스** | MongoDB 4 (Motor 비동기 드라이버) |
| **캐시/실시간** | Redis (캐싱, 잠금, 프레전스) |
| **AI 엔진** | Anthropic Claude API (Opus/Sonnet/Haiku) |
| **웹 검색** | Perplexity Search API |
| **문서 생성** | python-pptx, openpyxl, python-docx |
| **온라인 편집** | OnlyOffice Document Server |
